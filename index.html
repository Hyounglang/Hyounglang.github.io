三十岁的男人

Posted on 2018-05-19 | | Visitors:

       三十年来在自我认知的路上反复又反复，无常再无常，好在，我有点认识了自己。这三十年来，我对品行的总结是：坏的不彻底，好的不够安心；对一切感情的总结是：撕扯的时候太过粗暴，牵连的时候太过犹豫；对生活的总结是：人生多艰在于太多被动，往后我要追着它跑。

       三十年来我遇见了很多很多，我跨过江海、越过山川的时候它们都很安静，这有的时候令我失望，我穿过人流、透过喧嚣的时候，声音往往都扑面而来，把我挤在一个狭窄不堪的地方，这反而令我失去方向，我遇到很多的人，有人说我善良，有人说我做作，有人说我幼稚肤浅，有人说我成熟内敛，更有人告诉我应该去成为什么样的人。

       这三十年来我很庆幸，我庆幸我经历了一些东西，足够铭心，庆幸我能够及时的去回归自我，去反思自己的人生，体会这个世界，去认真的生活，认真的感知，认真的建立关于生活的信仰。我开始自己去定义幸福，去学着认真对待令人快乐的东西，也开始慢慢得到面对一切的勇气与信念。我只是开始有自己的态度了，往后要有脾气了。

        近些年来每一天，我都希望活成自己想要的样子，每天早晨，当我醒过来的时候，我面对的这个世界它已经开始了新的征程，它的改变令人无法料及，而我的时间有限，我该去做点什么。

TMD总要做点什么吧

Posted on 2018-05-19 | | Visitors:

       近来一直想要干点什么，程序员这个工作实在可恶，现在咱们是处于五年之痒的阶段。之前我们还有过两年三年之痒，每年痒一次。总之，我瞧它瞧的自己面目狰狞，实在干不长久，但又没有其他技长，所以就想要不创业吧，但是这个玩意绝不是简简单单的计划就行，囿于脾性又不肯草草起事。

       话说回来，抛弃程序员这个工作暂时是不可能发生的事情，出于它毕竟养活了我好几年，所以哪怕要分开也要找个最好的方式慢慢的来，浪漫告白，小心告别。

       后来又想，要不写书去吧，就是不知道遣词造句瞎搞搞、典故出处张冠李戴、出句就像走火一样的个性能不能找到哪个胡闹的刊社愿意出版。况且现在记性超级不得了的不行，有时候我就像《寻找无双》里宣阳坊的那一群坊民，轻易就把上一秒钟的事情忘的干干净净，而且想忘就忘。比如说我这一刻想去卫生间方便，然后下一秒我肩膀上就挂了条毛巾，神奇的是我还能正定自若的坐上坐便器，而不是去开水龙头，我简直像活在星际穿越四维空间里那个小伙子一样无奈，只不过这个空间现在在我的脑壳里。也许我就是缺了一串摩尔斯点码，不然上厕所和搭毛巾就有了合理的解释。后来又想这串狗屁码字原来其实是有的，不然解释不通原来怎么不会这样。也许是年少读书缺粮的时候给当了，连着身高也没长，好在没有横向发展，这说明天才有的是天赋去当，用不着庸人自扰。也是透过那段时光，发现原来自己是个喜欢乱丢东西的人。所以我在追逐时光步伐的时候步履并不轻盈更不稳健，不然何苦丢掉那么多。丢掉这些东西的后果就是要花上这辈子余下的时光再去收捡回来，这个时候我又仿佛成了个衣衫半褛的浪人，若不拾荒便于生活中难以为继。我总是有时间就会想想那些前半辈子舍弃的东西，人大都有个忆苦忆从前的毛病，而我更甚，我总觉得人这辈子唯一拥有的不是现在，更不是将来，它应该是值得记忆、值得铭刻的东西，这些东西有好的也有不好的，更有一些丢掉的“坏毛病”—在当下这个时刻却是非常非常特别的东西。但是有些东西丢也丢不掉，自恋这玩意就像牛皮糖一样，所以牛皮糖需要糖纸包着，也说明古人真的很了不起，研究发现了人需要穿衣服，现在总算明白了，不然人就像牛皮糖一样，稍微开点缝就要粘你一眼，所以在街上碰到开胸露腿的我总是偷偷的瞄两眼，从来不敢靠近，我瞄两眼是担心她们想耍阴谋诡计时刻要粘上来。

       大概我自己也知道，我就是个打鱼连网都不晒的人，所以写书这个想法最多也就TMD会持续那么一会，而且今后它将会不断随着某个时间出现，随着岁月此起彼伏不断跌宕的只会愈加凶狠，命运待它如待我一般不拘一格。说到这里代号甲的人会说那你还能干嘛，网都不晒的人不但无知，而且危险，更是地主主义表现。我说这又是为何啊？！这个时候他肯定不屑于回答，脸上写满了连这都不知道的样子。后来我算是搞明白了，说我危险是因为如果我不晒网肯定也懒得把网从湖里拉起来，这样整个鄱阳湖的鱼可能都要遭罪了，那样我可不就是个危险分子了吗……说我地主就更好解释了，一个靠打渔为生的人不晒网说明了不拉网，不拉网靠什么生活，这不说明了我就是个大财主嘛，我打渔肯定纯粹是为了消遣，消遣了鱼也消遣了其他的渔民，这样他们更加不高兴了，这要是老毛在的时候是要挨批的。所以就说明了做人要一板一眼，该是什么样还得是什么样，特立独行的人只适合云端行走。

       后来的后来啊，又想起一个职业也许适合我，百思之后，想想要不干脆去做个哲学家吧，青史留名还能影响众大，天下万民都是我的拥趸，打架超车再也不是势孤力单了，一个方向盘打下去历史就是一个大转弯，没准把成吉思汗给甩出来了，那样也给了他一个重开历史的机会，也没准这货会把尿撒到America，叫嚣宣告这块地盘以后就是老子的了。到时候说不准这货要到处养小老婆，子孙后代都繁衍到太平洋底下去了。这样一想又觉得不对，那样说来这货不就有可能成了我的祖先了嘛，而我又是纯正的汉族人氏，虽说跟着他混很光荣但是出卖先祖的事情还是不能干，再说这样一来不就乱套了嘛，所以我还是要规规矩矩一板一眼。虽然我很想穿越历史去告诉圣贤先生们他们都是肆意剽窃生灵的贼，一个近视一个白内障一个青光，所谓思想学说都是瞎扯，扯的还TMD都是一个东西，既然知道那么多难道还不知道怎么沉默吗。我想他们很需要一个明白人能够穿越历史去告诉他们事实的真相，这也算是我想成为哲学家的另一个原因吧，不然分量不足干不过他们。可是百思之后又多了一思，发现哲学家可能目标太大，不好做，容易被批判，而且说不定又有另一个我在盯着我要说我是沽名钓誉。这一思就好像一份一百分的卷子结果你考了一百零一分，老师就会想你这是多少年没考过试了？抱着宁可错杀的态度一定要把这一分给干下去，所以重批给了个一百分。这让我很忧郁，一百分的人不缺，一百零一分的人却世上少有，我既然要成为一个世界瞩目的人那说明我注定要成为一个少数分子，既然是少数分子怎么可以只考一百分。这说明如果最终我要是没有成为那少部分人，原因就在于老师扣了我的分数，这是命运对我的安排，此时的我还无法抗拒，只好默默忍受。就好像它在你必经之路上拉了一串八公分宽的屎，而这条路只有八公分宽（命运总是喜欢肆意捉弄，而且找的借口让你无可反驳，比如这个八公分，它会说你的鞋底宽度只有这么多，社会资源有限，况且还有的人在走钢丝，你要理解），但是我想总有一天我也可以肆意划分地界并且对它张牙舞爪令它无可奈何。这一天曾经有过，后来却没有再来，“这一天也许明天就来，这一天也许永远都不会再来。”

       所以，要不就退一步，做个伟人吧，伟人和哲学家相同的地方就是都能有足够多的影响；如果要说不同，大概就是哲学家能够在历史的漂流中被击碎后能还原于大地，受益于流窜的生命；而伟人据说能够“中流砥柱”，其实就是根石头做的二杆子，上面挂载几个红绿大灯，告诉历史你在我这里要转弯了，其实没他历史一样转弯。在我心里其实对这种结石一样的生物兴趣不大，但是如果你非得要选择一样职业，而且还要是少数人，所以要么是掌握部分真真假假的神人，要么做历史的转向灯，所以我的第二大目标也许就是成为一位结石吧，这更说明很多时候的恶心都是自寻烦恼。不过后来我学会了阿Q的精神胜利法，那就是如果那样的话，所有的路灯就都跟我很熟了，考个驾驶证也就容易得多了。

假装上班

Posted on 2018-05-19 | | Visitors:

       我现在坐在地铁上，昏昏沉沉的，身上挂着个小包，也摇摇晃晃的。我可能是去上班，地铁外面的天气有点热，这里的蚂蚁都不想出去，它们肯定是藏在哪里了，到了蚁后叫它们回家的时候才会爬出来。所以舒适的环境能让蚂蚁留恋，如果我是假装上班就好了，我就不用走出去了。

       自从没了寒暑假我已经忘了还有季节的存在，也混淆了天气的变化。我已经不记得春夏的转变了，如果这一天很热我们就叫它是夏，这一处阴凉我们就说这是春天来了，这样它们相互交替的日子就有了理由；此时我肯定是处在春天，如果我走出去，那肯定是要被烈日追索缠绕；所以我不想走出去，并不是因为我不想上班。而且现在的天气让我失去了辨认的能力，我只能靠着模糊的记忆：晴天是火辣的太阳，雨天是四溅的腥土味，雪天是……而现实是它们经常相互交媾，所以你要出来首先就需面对这样夸张的辨认问题，我很怂所以我也不想出来。

       如果我是去上班的话，这应该不是一趟单行的旅程，因为我总是要回来的，在这样普普通通的一天，至少我会这样记得。人如果远行，不管是要去到哪里，总是记得要回来，所以人是出不了家门的。
