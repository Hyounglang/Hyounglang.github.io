no coding

Posted on 2018-05-19 |

       我也不知道为什么坐在这里，假如我是一个时空穿越者，我肯定是从旧时代过来的，起点应该是近三十年前，他的轨迹该和如今的我相差不远，所以我没有多余的其他记忆，这样算来我只是多投了一次胎而已。我不知道为什么坐在这里，我发出这个问题是因为我面对着一团麻一样的东西，这段东西在这个时代有个专业的名词，叫做代码，英文名叫code，没猜错的话我应该是在coding。

       唯一能让人安慰的就是Lisa留下的风扇，这个风扇的作用仅能维持现在的胡思乱想，所以我现在不能coding。我记得自去年入秋以来，我就已经失去了coding的能力了。我进入了一个二次元coding的世界，往常的逻辑思维是不够用的，我之所以失去了这个能力，是因为我没有二次元的思维。后来我想通了，一个连初音是男是女都分辨不出来的人不配有二次元编码的能力。
